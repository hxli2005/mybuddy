MyBuddy：解压后双击

  1. 在 DeepSeek 开放平台 创建你自己的 API key，并确保账户有可用额度。
  2. 把整个 zip 解压到任意目录；中文名和空格都可以。不要直接在 zip 预览里运行。
  3. 双击 BuddyShell.exe，首次只粘贴一次 key，然后在聊天抽屉里对她说句话。
  4. 把小布拖到屏幕左边或右边再松手，她会栖在侧边；鼠标停近了才探头，点她或双击托盘图标就展开。 栖边只是身体摆放，不会被写成她的人生，也不会弹出主动气泡。
  5. 按住小布拖动，或拖动底部状态栏，可以真实提起并移动她；没有栖到侧边时，松开后她才会把这次动作当成发生过。
  6. 想换她会读的内容时，先退出，再编辑同目录的 小布读本.txt：保存为 UTF-8， 第一段写书名，正文段落之间留空行；换书时也换书名，她会从新读本开头开始。

对话会发给 DeepSeek，并由你自己的账户产生费用。 key 使用 Windows 当前用户加密，保存在 %APPDATA%\MyBuddy\settings.json； 她的记忆在 %APPDATA%\MyBuddy\mind。删除解压目录不会删记忆。

看到拦截或报错时

  • Defender/SmartScreen：小范围分享版未签名，PyInstaller 产物可能被误报。 只在你确认 zip 来源时，从 Windows 安全中查看“保护历史记录”并允许/还原；不确认来源就不要放行。
  • 8000 端口被占用：关闭占用该端口的本地程序，再重启小布。
  • key 错误：她会诚实显示“模型暂时不可用”；在“设置”里更换 key，然后重启。
  • 双击两次：第二次不会启动第二个写者，也不会建第二份记忆。
